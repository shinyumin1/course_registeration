► 시스템 유의사항

> 매크로 프로그램 방지

  • 매크로프로그램 사용을 방지하기 위하여 관리자가 지정한 횟수이상 저장을 시도하면 화면에 나타나는 문자열을 입력하고 계속 사용하여야 함.
  • 문자열 오류 횟수가 지정된 횟수 이상이면 자동 로그아웃되고 문자열 입력이 올바를 경우 이전 요청이 자동 수행됨

> 멀티 로그인 방지

  • 동일한 아이디로 여러 곳에서 로그인을 했을 경우 마지막 로그인만 유효하며 나머지는 로그아웃됨

> 팝업차단 해제

  • 팝업차단 설정이 해제되지 않았을 경우 팝업차단을 해제
  • 크롬(Chrome) : 우측 상단 메뉴 오픈 -> 설정 -> 설정검색창에 "팝업" 검색 -> 사이트 설정 -> 팝업 및 디렉션 -> 팝업 허용 또는 사이트 추가
  • 파이어폭스(Firefox) : 우측 상단 메뉴 오픈 -> 설정 > 개인정보 및 보안 > 팝업 창 차단 체크 해지 또는 예외 목록에 등록
  • 팝업 설정 및 해제는 브라우저, 버전별로 방법이 다를 수 있으니 해당 브라우저 공식사이트 또는 검색을 통해 확인 바람

> 모니터 해상도

  • 수강신청시스템은 1920*1080 해상도에 최적화되어 있으며 그 이상의 해상도로 화면 설정
  • 해상도가 작아 정상적으로 보이지 않을 경우 브라우저 확대, 축소 기능으로 사용자 환경에 맞게 조절 후 사용
  • 수강신청 전 미리 확인하여 수강신청 시 지장 없도록 유의하기 바람

> 브라우저

  • 크롬, 파이어폭스 브라우저에 최적화 되어있음
